Euclid's Elements

This site presents Dr. David E. Joyce's (Professor Emeritus, Clark University) online edition of Euclid's Elements — all thirteen books, 465 propositions, every diagram — with the figures now rendered live in the browser using the geomlib TypeScript library. Drag a point in any diagram and watch the construction follow.

This is a republication of Dr. Joyce's Elements with his permission, originally hosted at Clark University. See the about geomlib page for the library, the license, and other works powered by it.
